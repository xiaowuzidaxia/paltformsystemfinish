软件说明
1、本软件为设备指令收发测试版软件，目前只做过简单测试，可能存在大量未知bug。
2、本软件动态为动态编译，如需动态链接库，请自行寻找，如果找不到请联系我。
3、没有捕捉 ctrl+c信号，所以ctrl+c将无法优雅退出程序。可在命令行输入quit后ctrl+c退出。
4、网络连接后，按关闭软件也许不会正常关闭连接。如果确实无法正常关闭且需要正常关闭者，请在命令行输入quit后ctrl+c退出。
5、正式软件版本将会与此程序用用一个框架，在使用过程中如果遇到什么bug，请帮忙记录。感谢配合
6、该软件现有指令会延续到正式版本当中去，但是可能会做细微调整。


软件功能：
1、自带lua解释器，可加载执行lua脚本。
2、支持几个与设备通信的lua扩展函数
3、TCP链接设备，具体配置参看函数说明
４、支持现有协议的收发功能。
５、可通过命令行界面执行指令
